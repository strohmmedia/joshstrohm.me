**In short:** Fast automation wins deliver immediate time savings for SMBs by replacing repetitive tasks with simple, reliable flows.

## Identify

- Track time spent on recurring manual work for one week.
- Score tasks by frequency, time, and frustration (1–5 scale).
- Target items scoring 12+ for automation.

## Architect

- Use one integration per flow (Gmail → CRM, form → webhook, calendar → notes).
- Prefer native integrations or lightweight MCP/webhook bridges over custom scripts.
- Keep secrets in `.env` and use least‑privilege service accounts.

## Implement

1. Start with one high‑impact flow (e.g., lead intake or inbox triage).
2. Build and test in a sandbox.
3. Deploy with rollback capability.
4. Document runbooks and success metrics.

## Quick wins

- **Inbox → zero:** Rules + canned responses. Saves 2–4 hours/week.
- **Meeting notes:** Auto‑transcribe → summary → CRM. Saves 1–2 hours/week.
- **Lead intake:** Form → webhook → CRM + SMS. Faster response and 1–3 hours saved.

Automation is a discipline: start small, measure time saved, iterate, and expand.
